# AcFormer

## Code
This is the PyTorch implementation of our paper "AcFormer: An Aligned and Compact Transformer for Multimodal Sentiment Analysis", submitted to ACM MM 2023.

## Data
Data samples of six datasets (CMU_MOSI, CMU_MOSEI, UR-FUNNY, IEMOCAP, MUStARD, MELD): <https://drive.google.com/drive/folders/1o3onVsZsHpPAZZsY2G7ralSmL2jI508n?usp=sharing>
